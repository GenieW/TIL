## for(key in `OOO`)

- 배열

```sh
  const testArr = ["rk", "sk", "ek"];
  for(key in testArr) {
    console.log(key, testArr[key]);
  }
```

- 객체

```sh
  const testObj = {"rk":"가", "sk":"나", "ek":"다"};
  for(key in testObj) {
    console.log(key, testObj[key]);
  }
```
